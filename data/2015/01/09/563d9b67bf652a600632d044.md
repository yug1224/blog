突然 git が死んだ。

```
'git status' failed with code 6
```

git status をすると Error が表示される。

git log や git diff は出来るが、git add や git commit は怖くてやっていない。

`.git/index.lock`が出来ていたので、ひとまず削除。

```
$ rm .git/index.lock
$ git reset
```

なんとか git status が出来るようになった。
